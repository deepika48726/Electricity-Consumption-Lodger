Problem Description
This project is a console-based Java application designed to log electricity consumption records of users. It allows users to store and manage usage data efficiently. The system provides features to calculate total and average consumption. It also helps identify the highest electricity usage among records. The application is menu-driven and user-friendly.

Features
* Add electricity usage record
* View all records
* Calculate total consumption
* Calculate average consumption
* Find highest consumption
